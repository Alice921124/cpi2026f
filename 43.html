UVa 11321 客製化排序 (Sort! Sort!!)



給定 N 個數字與一個正整數 M，將這 N 個數字按照「除以 M 的餘數」由小到大排序。

若餘數相同，則依序套用以下規則打破平手（Tie-breaker）：
1. 奇數排在偶數前面。
2. 若皆為奇數，數字較大的排前面。
3. 若皆為偶數，數字較小的排前面。

⚠️ 注意：負數取餘數必須遵循 C 語言規則 (例：-100 % 3 = -1)。

輸入說明 輸出說明
1. 包含多組測資。
2. 每組第一行為 N 與 M。
3. 接著有 N 行數字。
4. 輸入以 "0 0" 結束。 		1. 先輸出 N 與 M。
2. 接著依序輸出排序後的 N 個數字。
3. 最後必須將結尾的 "0 0" 也印出來。

解答

副程式-compare(): 比較邏輯 

import sys
import math
from functools import cmp_to_key

# 宣告為全域變數，方便 compare 函數存取
M = 0

def compare(a, b):
    # 1. 取得 C 語言風格的餘數 (處理負數問題)
    # math.fmod 回傳浮點數，需轉為 int
    mod_a = int(math.fmod(a, M))
    mod_b = int(math.fmod(b, M))

    # 規則 1：餘數不同，餘數較小者排前面
    if mod_a != mod_b:
        return mod_a - mod_b

    # 判斷兩數是否為奇數 (不等於 0 即為奇數)
    a_is_odd = (a % 2 != 0)
    b_is_odd = (b % 2 != 0)

    # 規則 2：一奇一偶，奇數排前面
    if a_is_odd != b_is_odd:
        # 回傳負數代表第一個參數(a)排前面
        if a_is_odd:
            return -1
        else:
            return 1

    # 規則 3：兩者皆為奇數，數值大者排前面
    if a_is_odd and b_is_odd:
        return b - a

    # 規則 4：兩者皆為偶數，數值小者排前面
    if not a_is_odd and not b_is_odd:
        return a - b
主程式 

def main():
    global M # 宣告使用全域變數

    while True:
        try:
            line = input().split()
            if not line:
                continue
            N, M = map(int, line)

            # 遇到 0 0 代表結束，需印出 0 0
            if N == 0 and M == 0:
                print("0 0")
                break

            # 讀取 N 個數字
            nums = []
            for _ in range(N):
                nums.append(int(input()))

            # 呼叫 sort，透過 cmp_to_key 套用自定義的 compare 規則
            nums.sort(key=cmp_to_key(compare))

            # 輸出結果
            print(f"{N} {M}")
            for num in nums:
                print(num)

        except EOFError:
            break

if __name__ == "__main__":
    main()
