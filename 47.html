UVa 382 完美數判定 (Perfection)



給定一個正整數 N。

如果 N 的所有「正真因數（不含自身）」加總等於 N，則為完美數 (Perfect)。
若總和大於 N，為豐富數 (Abundant)。
若總和小於 N，為不足數 (Deficient)。

例: 數字 6 的真因數有 1, 2, 3。
1 + 2 + 3 = 6，因此 6 是完美數 (Perfect)。
例: 數字 9 的真因數有 1, 3。
1 + 3 = 4 (小於9)，因此 9 是不足數 (Deficient)。

輸入說明 輸出說明
1. 輸入包含一系列正整數。
2. 數字皆小於或等於 60,000。
3. 數字間以空白分隔，並以 0 作為結束標記。
 1. 第一行輸出 "PERFECTION OUTPUT"。
2. 中間針對每個數字輸出其分類。
 數字需靠右對齊佔 5 格，後接兩個空白與分類名稱。
3. 最後一行輸出 "END OF OUTPUT"。

Input/Output



Sample Input Output
15 28 6 56 60000 22 496 0 PERFECTION OUTPUT
   15  DEFICIENT
   28  PERFECT
    6  PERFECT
   56  ABUNDANT
60000  ABUNDANT
   22  DEFICIENT
  496  PERFECT
END OF OUTPUT

輸出排版提醒：請注意數字佔據了 5 個字元的寬度，且靠右對齊。
Python 可使用 f-string 的 {n:>5} 語法來達成。

解答

主程式 

            def main():
                # 1. 輸出開頭標語
                print("PERFECTION OUTPUT")
                
                while True:
                    try:
                        # 讀取一行輸入並用空白切割，轉換為整數列表
                        line = input()
                        if not line:
                            continue
                        nums = list(map(int, line.split()))
                        
                        # 2. 逐一處理列表中的數字
                        for n in nums:
                            if n == 0:
                                # 遇到 0 代表測資結束
                                print("END OF OUTPUT")
                                return
                            
                            # 3. 計算因數總和
                            divisors_sum = 0
                            # 迴圈範圍從 1 到 n 的一半 (無條件捨去)
                            for i in range(1, n // 2 + 1):
                                # 如果餘數為 0，代表 i 是 n 的因數
                                if n % i == 0:
                                    divisors_sum += i
                            
                            # 4. 判定性質與輸出
                            if divisors_sum == n:
                                result = "PERFECT"
                            elif divisors_sum < n:
                                result = "DEFICIENT"
                            else:
                                result = "ABUNDANT"
                                
                            # 5. 格式化輸出：:>5 代表靠右對齊並佔 5 格
                            print(f"{n:>5}  {result}")
                            
                    except EOFError:
                        break

            if __name__ == '__main__':
                main()
