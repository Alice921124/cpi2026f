41 CPE 踩地雷


遊戲的目標是在一個 M × N 的地圖中找出所有地雷的位置。為了協助玩家，每一個不是地雷的格子都會顯示一個數字，表示與該格相鄰的地雷數量。
例如，下列是一個 4 × 4 的地圖，其中有兩顆地雷（以 * 表示）：

*...
....
.*..
....

若將每個安全格改為周圍相鄰地雷的數量，則會得到：

*100
2210
1*10
1110

每一個格子最多會有 8 個相鄰的格子。

Input/Output

輸入說明 輸出說明
輸入包含多組測試資料。

每組測試資料的第一行包含兩個整數 n 和 m（0 < n, m ≤ 100），分別代表地圖的列數（Rows）與行數（Columns）。

接下來共有 n 行，每一行包含 m 個字元，表示整張地圖。
.（句點）代表安全格。
*（星號）代表地雷。

當輸入為 0 0 時，表示所有測試資料結束，該組資料不需要進行處理。 		對於每一組測試資料，請先輸出一行：
Field #x:
其中 x 代表目前是第幾組測試資料（從 1 開始編號）。

接著輸出處理後的地圖：
原本是 * 的位置保持不變。
原本是 . 的位置改為該格周圍八個方向相鄰地雷的數量。

每兩組測試資料之間必須空一行。

解答

主程式 

            def main():

                case = 0
                while True:
                    #讀入 n,m
                    n, m = map(int, input().split())

                    #如果 n=m=0 則結束
                    if n == 0 and m == 0:
                        break

                    #建立地圖
                    case += 1
                    field = []
                    for i in range(n):
                        field.append(list(input()))

                    #輸出答案
                    print("Field #%d:" % case)

                    #逐格檢查
                    for i in range(n):
                        for j in range(m):

                            #如果是地雷，輸出 *
                            if field[i][j] == '*':
                                print("*", end="")

                            #否則，計算八個方向地雷數
                            else:
                                count = 0
                                for dx in [-1, 0, 1]:
                                    for dy in [-1, 0, 1]:
                                        if dx == 0 and dy == 0:
                                            continue
                                        x = i + dx
                                        y = j + dy
                                        if 0 <= x < n and 0 <= y < m:
                                            if field[x][y] == '*':
                                                count += 1

                                print(count, end="")

                        print()

                    print()
